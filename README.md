# 🚀 Como começar?

```sh
  git clone https://github.com/ogabrielrodrigues/ignitelab.git
  cd ignitelab
```

```sh
  npm i # com NPM
  yarn # com Yarn
```

```sh
  npm run dev # com NPM
  yarn dev # com Yarn
```

## O projeto está disponivel em: _*http://localhost:3000*_
